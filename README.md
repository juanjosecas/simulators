# Simulators Repository

This repository brings together small simulation projects built to explore scientific and technical ideas in a practical way.

The main purpose of this space is experimentation. Each simulator is a focused environment where an idea can be tested, visualized, and improved through iteration. The projects are independent, but they share the same intention: turning abstract concepts into something interactive and understandable.

This repository is designed to stay open and adaptable. New simulators can be added over time, existing ones can evolve, and the structure can grow with the needs of the work. It is both a working collection and a learning process.

## Repository Structure

Each top-level folder contains a simulator or related resource.

Current examples include:

- `electrophoresis_sim`
- `liquid_extraction_simulator`
- `pk-simulator`
- `tlc_simulator`

## Contribution and Evolution

Contributions are welcome when they align with the spirit of the repository:

- Keep simulators focused and understandable.
- Prefer clear structure over unnecessary complexity.
- Document enough context so others can continue the work.

This project exists to support exploration, teaching, and continuous improvement through simulation.
